1. Executive Summary & Architecture Context
The current digital infrastructure for the Puducherry Regional Transport Office (RTO) functions via a hybrid model. The local state portal operates primarily as a static informational index, while all core, transactional user workflows are executed via redirections to the central government's National Parivahan Sewa portal (Sarathi for licensing, Vahan for vehicle logistics). This report documents current feature efficacy and details the structural plans for a completely modernized portal.
The Redirection UX Trap: The primary systemic friction point stems from design and session discontinuity. Users initiating local requests are transferred unexpectedly to the massive national infrastructure, disrupting user flows and reducing application trust markers.

4.2 The Violation Scoring State Machine
The frontend tracker handles browser events and facial embeddings dynamically using a continuous grace period algorithm to prevent false flags from bad environmental illumination:
· Immediate Violations: Evading mandatory Fullscreen or switching windows logs an immediate penalty value score of 3.
· Biometric Validation: Webcam snapshots are processed every 3 seconds. Continuous matching exceptions (No Face / Face Mismatch) require 3 sequential failures before pushing an active violation penalty of 2.
· Termination Condition: If the total session severity score calculations match or exceed 5, the engine calls a background mutation API to immediately terminate input capability and mark the test session status as corrupted.
