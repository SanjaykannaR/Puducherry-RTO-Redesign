# Contact Page

**File:** `src/app/contact/page.tsx`

## Purpose
Contact information page with office address, helpline numbers, contact form, and regional office details.

## Key Decisions
- Client-side contact form (Phase 1; backend integration in Phase 2)
- Helpline numbers grouped by service type
- Regional offices displayed in cards for quick reference
- Required field indicators with red asterisk (WCAG 2.1 AA)

## Related Files
- `backend/src/routes/info.ts` - API endpoint for contact data
- `backend/src/routes/directory.ts` - API endpoint for office directory
